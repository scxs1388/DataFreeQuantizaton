# Data-Free Quantization
Data-Free (or Zero-Shot) Quantization is a method to quantize a deep neural network model without any original training data. It is a very challenging topic in the field of model compression. This page is a collection of papers and baselines on data-free quantization research. If you have any suggestions, please feel free to contact me.

(↑Generated by GitHub Copilot↑)

# Introduction
There are two branches of data-free quantization methods: theoretical analysis and data generation. 

## Theoretical Analysis
- Weight and activation distribution analysis for better clipping threshold.
- Quantization bias correction in case of the accumulation of quantization error throughout the network.
- Approximate the data-dependent task loss with data-free local loss. 

## Data Generation
- Data distillation: train batches of data samples and use them for model quantization.
- Adversarial: train a data generator in an adversarial manner to generate data samples and fine-tune the quantized model with knowledge distillation.
